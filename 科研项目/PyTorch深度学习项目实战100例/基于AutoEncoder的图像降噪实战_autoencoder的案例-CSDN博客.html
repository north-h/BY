一、基于AutoEncoder的图像降噪

对于生活中我们有一些图像数据是含有噪声的，或者在一些应用当中我们的图像太大，所以为了不过多影响图像中的数据信息，我们可以使用深度学习算法进行压缩，将原始图像数据映射到一个新的维度空间，然后再从这个维度空间复原到原来的空间，从而达到图像降噪的目的。

本项目是使用AutoEncoder自编码器来实现的图像降噪操作，实现一个简单的图像降噪实战案例。

二、AutoEncoder模型

自编码器（autoencoder, AE）是一类在半监督学习和非监督学习中使用的人工神经网络（Artificial Neural Networks, ANNs），其功能是通过将输入信息作为学习目标，对输入信息进行表征学习（representation learning）。

在深度学习中，自动编码器是一种 无监督 的 神经网络 模型，它可以学习到输入数据的隐含特征，这称为编码(coding)，同时用学习到的新特征可以重构出原始输入数据，称之为解码(decoding)。

从直观上来看，自动编码器可以用于 特征降维 ，类似主成分分析 PCA ，但是其相比 PCA 其性能更强，这是由于神经网络模型可以提取更有效的新特征。除了进行特征降维，自动编码器学习到的新特征可以送入有监督学习模型中，所以自动编码器可以起到 特征提取器 的作用。

三、定义网络结构

  • Encoder编码器：从原始数据中提取信息，将高级特征映射到一个新的维度空间，进行降维
  • Decoder解码器：将中间的隐藏空间重新映射成原来的样本空间，形成去噪后的图像数据
class AutoEncoder(nn.Module):
    def __init__(self):
        super(AutoEncoder, self).__init__()
        # 编码器
        self.encoder1 = nn.Linear(32*32*3, 1024)
        self.encoder2 = nn.Linear(1024, 512)
        # 解码器
        self.decoder1 = nn.Linear(512, 1024)
        self.decoder2 = nn.Linear(1024, 32*32*3)
        
    def forward(self, x):
        x = torch.flatten(x, start_dim=1)
        x = self.encoder1(x)
        x = self.encoder2(x)
        x = self.decoder1(x)
        x = self.decoder2(x)
        
        return x

五、定义损失函数

由于AutoEncoder的输入是属于自监督模型，模型的输入是图片，模型的输出是去噪后的图片，所以我们希望两张图片尽可能相近，所以我们会将整张图像的像素展开，然后计算MSE损失。

# 自定义损失函数，需要在forward中定义过程
class MyLoss(nn.Module):
    def __init__(self):
        super(MyLoss, self).__init__()
    
    # 参数为传入的预测值和真实值，返回所有样本的损失值，自己只需定义计算过程，反向传播PyTroch会自动记录，最好用PyTorch进行计算
    def forward(self, output_img, input_img):
        # pred：[32, 4] label：[32, 1] 第一维度是样本数
        
        return F.mse_loss(input_img, output_img)

import torchvision
from torch import nn
import numpy as np
import pandas as pd
import os
import json
import pickle

import torch
import torch.optim as optim
import torch.nn.functional as F
from torchvision import transforms, datasets
import torchvision.models as models
from torch.utils.data import Dataset, DataLoader
from tqdm import tqdm
from PIL import Image
from functools import partial
from collections import OrderedDict

import matplotlib.pyplot as plt

image_path = './data' # 数据集路径
epochs = 300
lr = 0.003
batch_size = 32
save_path = './best_model.pkl'

device = torch.device('cuda:0' if torch.cuda.is_available() else 'cpu')

# 1.数据转换
data_transform = {
    # 训练中的数据增强和归一化
    'train': transforms.Compose([
        transforms.RandomResizedCrop(32), # 随机裁剪
        transforms.RandomHorizontalFlip(), # 左右翻转
        transforms.ToTensor(),
        transforms.Normalize([0.485, 0.456, 0.406], [0.229, 0.224, 0.225]) # 均值方差归一化
    ])
}

# 2.形成训练集
train_dataset = datasets.ImageFolder(root=os.path.join(image_path),
                                     transform=data_transform['train'])


# 3.形成迭代器
train_loader = torch.utils.data.DataLoader(train_dataset,
                                           batch_size,
                                           True)

print('using {} images for training.'.format(len(train_dataset)))

# 自定义损失函数，需要在forward中定义过程
class MyLoss(nn.Module):
    def __init__(self):
        super(MyLoss, self).__init__()
    
    # 参数为传入的预测值和真实值，返回所有样本的损失值，自己只需定义计算过程，反向传播PyTroch会自动记录，最好用PyTorch进行计算
    def forward(self, output_img, input_img):
        # pred：[32, 4] label：[32, 1] 第一维度是样本数
        
        return F.mse_loss(input_img, output_img)

class AutoEncoder(nn.Module):
    def __init__(self):
        super(AutoEncoder, self).__init__()
        # 编码器
        self.encoder1 = nn.Linear(32*32*3, 1024)
        self.encoder2 = nn.Linear(1024, 512)
        # 解码器
        self.decoder1 = nn.Linear(512, 1024)
        self.decoder2 = nn.Linear(1024, 32*32*3)
        
    def forward(self, x):
        x = torch.flatten(x, start_dim=1)
        x = self.encoder1(x)
        x = self.encoder2(x)
        x = self.decoder1(x)
        x = self.decoder2(x)
        
        return x

# 定义AutoEncoder模型
model = AutoEncoder()

# 将模型加载到cpu中
model = model.to('cpu')

criterion = MyLoss() # 损失函数
optimizer = torch.optim.Adam(model.parameters(), lr=lr) # 优化器

# 6.模型训练
best_loss = 999 # 最优精确率
best_model = None # 最优模型参数

for epoch in range(epochs):
    model.train()
    running_loss = 0 # 损失

    train_bar = tqdm(train_loader)
    for data in train_bar:
        images, labels = data
        optimizer.zero_grad()
        output = model(images.to(device))
        loss = criterion(output, torch.flatten(images, start_dim=1))
        loss.backward()
        optimizer.step()

        running_loss += loss.item()
        train_bar.desc = "train epoch[{}/{}] loss:{:.3f}".format(epoch + 1,
                                                                 epochs,
                                                                 loss)
       
    # 打印信息
    print("【EPOCH: 】%s" % str(epoch + 1))
    print("训练损失为%s" % str(running_loss))

    if running_loss < best_loss:
        best_loss = running_loss
        best_model = model.state_dict()

    # 在训练结束保存最优的模型参数
    if epoch == epochs - 1:
        # 保存模型
        torch.save(best_model, save_path)

print('Finished Training')
